Local Verification of CAN Interface

In this video, we set the recieve filter for values between 0x100 and 0x103. First, we send a message with ID 0x104 which is transmitted but not received. Then we send a message with ID 0x103 which is transmitted and received (hits a breakpoint between transmit and receive so the LED cycles are distinct).
